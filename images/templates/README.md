# 템플릿 미리보기 이미지 가이드

## 📁 파일 구조
```
images/templates/
  ├── sns-minimal.png
  ├── sns-modern.png
  ├── shopping-minimal.png
  └── ...
```

## 📐 권장 사이즈
- SNS 템플릿: 400x500px
- 쇼핑몰 상세페이지: 600x800px
- 광고 배너: 1200x400px
- 이메일 템플릿: 600x900px

## 💡 이미지 생성 방법
1. Figma에서 템플릿 스크린샷
2. 포토샵/Figma로 편집
3. PNG/JPG로 저장 (최적화)
4. 이 폴더에 업로드

## 🔗 templates.json에서 사용
```json
"previewImage": "/images/templates/sns-minimal.png"
```
